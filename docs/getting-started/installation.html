Installation

This guide will help you install and set up the TS-MCP-Client in your project.

Prerequisites

Before installing the TS-MCP-Client, make sure you have:

  • Node.js (version 16 or higher)
  • npm or yarn
  • TypeScript (for TypeScript projects)
  • API keys for at least one LLM provider (Anthropic, OpenAI, etc.)

Installation Steps

1. Install the package

Add the package to your project using npm:

npm install @rinardnick/client_mcp@1.2.1

Or using yarn:

yarn add @rinardnick/client_mcp@1.2.1

2. Create a configuration file

Create a configuration file (e.g., mcp-config.json) with your provider information: 

{
  "providers": {
    "anthropic": {
      "api_key": "your-anthropic-api-key",
      "default_model": "claude-3-sonnet-20240229",
      "system_prompt": "You are a helpful AI assistant..."
    }
  },
  "default_provider": "anthropic"
}

For security reasons, it's recommended to use environment variables for API keys: 

{
  "providers": {
    "anthropic": {
      "api_key": "${ANTHROPIC_API_KEY}",
      "default_model": "claude-3-sonnet-20240229",
      "system_prompt": "You are a helpful AI assistant..."
    }
  },
  "default_provider": "anthropic"
}

3. Import and initialize the client

Create a new file (e.g., client.ts or client.js) to initialize the client: 

import { SessionManager, loadConfig } from '@rinardnick/client_mcp';
import path from 'path';

// Load and process configuration
const config = loadConfig(path.resolve('./mcp-config.json'));

// Initialize the session manager
export const sessionManager = new SessionManager(config);

// Export a function to create a new session
export async function createSession() {
  return await sessionManager.createSession();
}

4. Verify installation

Create a simple test script to verify the installation:

import { sessionManager, createSession } from './client';

async function testClient() {
  try {
    // Create a new session
    const sessionId = await createSession();
    console.log('Session created:', sessionId);
    
    // Send a test message
    const response = await sessionManager.sendMessage(
      sessionId, 
      'Hello, can you confirm this is working?'
    );
    
    console.log('Response:', response.content);
    
    // Close the session
    await sessionManager.closeSession(sessionId);
    console.log('Session closed successfully');
  } catch (error) {
    console.error('Error testing client:', error);
  }
}

testClient();

Run the test script:

node -r ts-node/register test.ts
