TS-MCP-Client

A TypeScript implementation of the Model Context Protocol (MCP) client that manages LLM chat interactions, server lifecycle, and tool invocations with multi-provider support.

Get Started

Why TS-MCP-Client?

TS-MCP-Client simplifies the integration of powerful LLM capabilities into your applications by handling complex interactions with LLM providers and tools.

  • Support for multiple LLM providers (Anthropic, OpenAI, Grok)
  • Tool integration for enhanced AI capabilities
  • Session management and persistence
  • Token optimization and context management
  • Seamless provider switching during conversations
import { SessionManager, loadConfig } from '@rinardnick/client_mcp';

// Load configuration
const config = loadConfig('./config.json');

// Initialize the session manager
const sessionManager = new SessionManager(config);

// Create a new session
const sessionId = await sessionManager.createSession();

// Send a message and get response
const response = await sessionManager.sendMessage(
  sessionId, 
  'Hello, how are you today?'
);

console.log(response.content);

Getting Started

Installation

npm install @rinardnick/client_mcp

Add the package to your project using npm or yarn.

Basic Configuration

{
  "providers": {
    "anthropic": {
      "api_key": "your-anthropic-api-key",
      "default_model": "claude-3-sonnet-20240229"
    }
  },
  "default_provider": "anthropic"
}

Create a configuration file with your provider API keys.

Examples

Basic Conversation

// Create a session
const sessionId = await sessionManager.createSession();

// Send messages
await sessionManager.sendMessage(
  sessionId, 
  "What is the capital of France?"
);

// Close when done
await sessionManager.closeSession(sessionId);

Streaming Response

const stream = await sessionManager.sendMessageStream(
  sessionId, 
  "Write a short story about a robot"
);

for await (const chunk of stream) {
  if (chunk.type === 'content') {
    process.stdout.write(chunk.content);
  }
}
